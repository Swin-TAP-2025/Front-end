# mg4
Working with VScode vue3
## Project setup
```
npm install -g @vue/cli
npm install axios
```
Create project on VScode terminal
```
vue create [project name]
```
### Compiles and hot-reloads for development
```
cd your project folder
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
